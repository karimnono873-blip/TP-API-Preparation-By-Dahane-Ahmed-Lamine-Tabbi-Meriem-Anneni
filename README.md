## ✨ Fonctionnalités Principales

1. **Simulateur LADDER Intégré (Partie 1) :** * Moteur de rendu logique en pur JavaScript/CSS reproduisant le cycle de scrutation d'un CPU S7-300.
   * Interaction en temps réel avec les capteurs virtuels (CPT_LSL, CPT_LSH).
   * Visualisation dynamique du flux de courant (Power Flow) dans les réseaux LADDER.
   * Simulation visuelle du bloc de temporisation `TON` (SFB4).
2. **Tables d'Affectation des E/S :** Cartographie mémoire complète et détaillée pour le câblage physique des modules DI/DO.
3. **Analyse GRAFCET (Partie 2) :** Décomposition d'un poste d'usinage complexe avec tâches simultanées (Divergences en ET).
4. **Guide de Déploiement :** Procédure pas à pas pour la configuration matérielle sous **TIA Portal V21** et l'utilisation de **S7-PLCSIM**.
